# Restaurant Rating Prediction using Machine Learning

## Overview
This project predicts restaurant ratings using machine learning models.

## Models Used
- Random Forest
- Gradient Boosting
- Ridge Regression

## Results
- Best Model: Gradient Boosting
- R² Score: ~0.95

## Features
- Cost
- Votes
- Price Range
- Table Booking
- Online Delivery

## Outputs
- Model metrics (CSV)
- Visualizations (PNG)
- Summary report

## Tech Stack
Python, Pandas, Scikit-learn, Matplotlib
