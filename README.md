#  Restaurant Rating Prediction (Machine Learning)

## Overview
This project builds an end-to-end machine learning pipeline to predict restaurant ratings based on cost, votes, and service features.

## ⚙️ Models Used
- Random Forest
- Gradient Boosting  (Best)
- Ridge Regression

##  Results
- Best Model: Gradient Boosting
- R² Score: ~0.95
- Low prediction error (MAE & RMSE)

##  Key Insights
- Votes and cost strongly influence ratings
- Online delivery and booking improve ratings
- Tree-based models outperform linear models

##  Outputs
- model_metrics.csv
- results_summary.txt
- Visualizations

##  Tech Stack
Python, Pandas, Scikit-learn, Matplotlib

##  Future Improvements
- Hyperparameter tuning (GridSearchCV)
